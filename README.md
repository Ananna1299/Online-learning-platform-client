Website Name: eLearning

Features:
1. Users can filter the courses by course category.
2. Logged in users can view the details of the courses.
3. Logged in users can add new course.
4. Logged in users can view his/her added courses.
5. Logged in users can enroll course and view them.
6. Logged in users can delete their added courses. 
7. Logged in users can update their added course information.
8. Theme can be toggled.
