Website Name: eLearning

Overview: The platform enables users to browse and filter courses by category, view detailed course information, and manage their own courses. Logged-in users can add, update, or delete courses, enroll in courses, view enrolled courses, and toggle between themes.

Key Features:
1. Users can filter the courses by course category.
2. Logged-in users can view the details of the courses.
3. Logged-in users can add a new course.
4. Logged-in users can view their/added courses.
5. Logged-in users can enroll in courses and view them.
6. Logged-in users can delete their added courses. 
7. Logged-in users can update their added course information. 8. Theme can be toggled.

Live Link: https://online-learning-platform-13955.web.app/

Dependencies: @tailwindcss/vite", "aos", "firebase", "framer-motion", "react", "react-dom", "react-icons", "react-router", "react-toastify", "sweetalert2", "tailwindcss

Running the Project Locally:

Clone the repository: git clone repo link
Navigate to the project folder: cd project-folder
Install dependencies: npm install
Set up environment variables: ->Create a .env file in the root directory. ->Add any required keys, for example: VITE_API_KEY=your_api_key_here, VITE_FIREBASE_CONFIG=your_firebase_config_here
Start the development server: npm run dev
